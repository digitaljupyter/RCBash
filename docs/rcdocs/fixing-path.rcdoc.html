Fixing RCBash Path Issues

RCBash is written in Python, and uses local libraries to write data specially.

The way to use the RCBash.py globally, is by setting PYTHONPATH to the directory of RCBash.

Example:

export PYTHONPATH=$PYTHONPATH:/home/YOU/RCBash

This will allow RCBash to run the "RCScript" Library (a dependency for RCBash, RCBash is just the loader, RCScript is the official base lib)
